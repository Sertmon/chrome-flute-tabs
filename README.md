# Флейта — табы

Chrome-расширение: слушает микрофон, определяет ноты и показывает аппликатуру **сопрано блокфлейты в C**.

Подходит для игры на флейте и для напевания мелодии. Запоминает **имя ноты** (до, ре, ми…) без октавы — как в песне. Аппликатура подставляется на ближайшую октаву блокфлейты.

## Установка

1. Откройте `chrome://extensions`
2. Включите **Режим разработчика**
3. **Загрузить распакованное** → выберите эту папку (`Флейта`)

## Использование

1. Нажмите на иконку расширения — откроется **боковая панель**
2. **Слушать** — разрешите микрофон
3. Играйте на флейте или напевайте
4. Держите ноту ~полсекунды, между нотами — короткая пауза
5. Записанные ноты и аппликатура появятся в списке ниже

Кнопки: **Стоп**, **Очистить**, **Убрать последнюю**.

Если микрофон не работает — **Разрешения расширения** или **Открыть во вкладке** в панели.

## Структура

```
Флейта/
  manifest.json
  background.js
  sidepanel/     — основной интерфейс
  app/           — запасной интерфейс во вкладке
  lib/           — pitch (YIN), ноты, аппликатура, запись
```

## Техническое

- Определение высоты: алгоритм **YIN**
- Детектор: ~100–2400 Hz (низкий напев и флейта)
- Запись: до / ре / ми… без номера октавы
- Табы: сопрано C, нота переносится на ближайшую октаву флейты
