# Флейта — табы

Chrome-расширение: слушает микрофон, определяет ноты и показывает аппликатуру **сопрано блокфлейты в C**.

Подходит для игры на флейте и для напевания мелодии. Запоминает **имя ноты** (до, ре, ми…) без октавы — как в песне. Аппликатура подставляется на ближайшую октаву блокфлейты.

## Установка

1. Откройте `chrome://extensions`
2. Включите **Режим разработчика**
3. **Загрузить распакованное** → выберите эту папку (`Флейта`)

## Использование

### Микрофон
1. Источник звука: **Микрофон**
2. **Слушать** — разрешите микрофон
3. Играйте, напевайте или свистите

### Звук вкладки (YouTube и др.)
1. Откройте **YouTube** с видео и **включите воспроизведение**
2. На **этой вкладке** нажмите **иконку расширения** (боковая панель)
3. Источник **Звук вкладки** → **Слушать**

Звук берётся **напрямую с вкладки** (как в расширении Shazam) — диалог «Поделиться экраном» не нужен. Работает и в наушниках: микрофон не используется.

Пока панель открыта, **не нажимайте иконку повторно** — она закроет панель.

Монитор: шкала центов, пианино (по умолчанию C2–C8), история мелодии.  
Записанные ноты (до/ре/ми) и аппликатура — в списке ниже.

Лучше всего работает **одна флейта** в записи без громкого аккомпанемента.

Кнопки: **Стоп**, **Очистить**, **Убрать последнюю**. Настройки сохраняются автоматически.

Если микрофон не работает — **Разрешения расширения** или **Открыть во вкладке** в панели.

## Структура

```
Флейта/
  manifest.json
  background.js
  sidepanel/     — основной интерфейс
  app/           — запасной интерфейс во вкладке
  lib/           — pitch (YIN), ноты, аппликатура, запись
```

## Техническое

- Определение высоты: алгоритм **YIN**
- Детектор: по умолчанию ~100–4200 Hz (напев, свист, флейта)
- Пианино: нижняя/верхняя октава в **Настройках** (для свиста — до C8 и выше)
- Запись: до / ре / ми… без номера октавы
- Табы: сопрано C или альт F, октава для аппликатуры настраивается
- Все параметры детектора и записи — в блоке **Настройки**
